将错误信息输出到 log 文件 ./program 2>log，这样，标准输出显示在屏幕，但是错误信息会输出到log 文件中。

表情

github 中的表情都以 :+表情对应的单词+: 开头，加入表情对应的英文。例如

  • 向上的箭头 :arrow_up:
  • 100分 ‘:100:’

输入后预览即可看见表情，如果在 github 网页上输入 : 会有自动补全的哟。

将鼠标停放在图标上，也可以看到对应的表情值。
