Windows 必须要做的事情

注意，一定要开启 VT-x/AMD-V 硬件加速

开机进入BIOS选项 ，依次选Config->CPU->Intel Virtualization Technology，里面有个Intel VT-d Feature ，改成Enabled ，保存退出，关机，然后启动机器。
