atom 编辑器

发表于 2016-07-15   |   分类于 web   |  

Atom 编辑器使用 Web技术（html、css、javascript）开发。

详情

默认支持 markdown 预览功能。

自动换行

Atom默认会根据窗口宽度对文本进行自动软换行处理，如果没有的话请勾选：-

  • Atom->Preferences->Settings->Soft Wrap
  • Atom->Preferences->Settings->Soft Wrap At Preferred Line Length

常用插件

Cmd+,

  • file-icons
  • docblockr，注释，使用方法，/** + Tab
  • emmet
  • react-es6-snippets,
    • rcc + rbm
    • rfunc
  • advanced-open-file
  • open-in-browser
