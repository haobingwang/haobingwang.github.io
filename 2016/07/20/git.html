https协议 和 git协议

以 gitlab 为例：在服务器上进行 git fetch 等操作时。

  • 以 https 协议的仓库 url 作为远程仓库时会要求手动输入用户名和密码(只需输入一次)。
  • 而 git 协议的仓库 url 作为远程仓库却不需要输入用户名和密码，只要添加了 ssh公钥。
