始化设置

这里需要打开几个选项:

  • Expand Filesystem 执行该动作可以将根分区扩展到整张SD卡;
  • Change User Password 可以在这里修改默认密码;
  • Enable Boot to Desktop/Scratch 选择默认的启动模式，是图形界面还是文本模式;
  • Internationalisation Options 配置当前的
    • Locale,选择en_US.UTF-8和zh_CN.UTF-8;
    • Timezone,选择Asia – Shanghai;
    • Keyboard Layout,选English（US）;
  • Enable Camera 是否启用树莓派摄像头功能;
  • Add to Rastrack 是否要加入树莓派地图，一般不添加;
  • Overclock 超频设置，自己按要求设置，稳定第一，一般800MHz，900MHz问题不大;
  • Advanced Options
    • Overscan 可以扩充或者缩小屏幕的设置。
    • Hostname 设置树莓派的主机名，方便网络访问时查看。
    • Memory Split 设置分配多少内存给GPU使用。
    • SSH 是否启用SSH远程访问功能,请务必启用该功能（默认开启）。
    • Device Tree 是否启用设备树功能。
      • CPU的数量和类别
      • 内存基地址和大小
      • 总线和桥
      • 外设连接
      • 中断控制器和中断使用情况
      • GPIO控制器和GPIO使用情况
      • Clock控制器和Clock使用情况
  • SPI 启用或者禁用自动加载SPI(串行外设接口)内核模块，需要其它的外设产品例如PiFace。
  • I2C 是否自动加载I2C内核模块
  • Serial 是否启用shell与串口通信功能，如果要使用GPIO，建议启用。
  • Audio 强制音频通过HDMI或者3.5毫米接口输出。
  • Update 将这个工具升级到最新版本。

设置完成后，选择Finish，会提示是否重启，选择Yes，就会重启

解决语系错误

安装软件时遇到如下错误：

原因是远端(树莓派)会读取本地(OSX)上的语系配置(zh_CN.UTF-8) 但远端(树莓派)支持语系里没有本地当前语系，所以出现上述错误。

当前语系配置：locale
当前支持语系：locale -a

结果中没有 zh_CN.UTF-8,所以需要下载

sudo apt-get install language-pack-zh

或者

sudo locale-gen zh_CN.UTF-8

现在问题就可以解决了。

step1:格式化SD卡

首先格式化磁盘。sdcard-formatter，这个只识别sd卡，不会出现格式错误的问题。支持 windows 和 mac 两个系统。

  • 覆盖格式化
  • 调整逻辑大小

或者，使用 Mac 自带的磁盘工具，左侧选中对应的SD卡，点击“抹掉”

  • 格式选择“MS-DOS(FAT)”

step2:查看挂载的卷

Identify the disk (not partition) of your SD card e.g. disk2, not disk2s1.

step3:分区卸载

Unmount the partition so that you will be allowed to overwrite the disk. 如果不卸载分区，直接 dd 就会提示 dd: /dev/disk2: Resource busy

Unmount of all volumes on disk2 was successful

step4:写入系统镜像

Copy the data to your SD card.

Be sure to use /dev/rdiskn instead of just /dev/diskn. This way you are not writing to a buffered device and it will complete much faster.

(说明：/dev/disk1s1是分区，/dev/disk1是块设备，/dev/rdisk1是原始字符设备)

This will take a few minutes, depending on the image file size. You can check the progress by sending a SIGINFO signal (press Ctrl+T).

rdisk vs disk

To get the best speed out of your MacBook’s SD slot, you need to use a ‘raw disk’ device.

This is a SanDisk Ultra Class 10 card:

~/Desktop/rpi $ sudo dd if=2012-12-16-wheezy-raspbian.img of=/dev/disk1 bs=8m
231+1 records in
231+1 records out
1939865600 bytes transferred in 1076.078932 secs (1802717 bytes/sec)

Compare this to using /dev/rdisk1 instead of /dev/disk1

~/Desktop/rpi $ sudo dd if=2012-12-16-wheezy-raspbian.img of=/dev/rdisk1 bs=1m
1850+0 records in
1850+0 records out
1939865600 bytes transferred in 129.712100 secs (14955163 bytes/sec)

1.8MB/s compared to 14.9MB/s - almost 10x the improvement!
