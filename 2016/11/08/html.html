块级元素 && 行内元素

块级元素占据其父元素（容器）的整个空间。
行内元素只占据它对应标签的边框所包含的空间。

汉字占位符问题

一个汉字使用一个全角汉字空格。搜狗输入法的默认是半角，所以需要手动更改下设置再输入空格即可。使用空白字符会被 htmlmin 等工具剔除掉。推荐使用汉字占位符 &#12288;

客户端存储

远古时期

  • cookies 的用法和缺陷
  • userdata

cookie

  • 每次 http 请求都会带上 cookie 跟服务器要东西（不安全、请求臃肿）
  • 最大存储 4K
  • 主域名污染，子域名会受主域名影响

userdata

  • 只有 IE 支持
  • XML 文件

HTML5 时代:

H5 存储的主要目标是：

  • 解决 4K 限制问题
  • 解决请求头常带存储信息问题
  • 解决关系型存储的问题
  • 解决跨浏览器的问题
  • LocalStorage
    • 存储形式：键值对
    • 存储容量：每个域名 5M
    • 过期：永不过期，手动删除
  • SessionStorage
    • 过期：关闭页面，或重新打开浏览器，或者重新打开新Tab，刷新不会过期
  • IndexedDB
  • Web SQL（已不再维护）
  • application cache
