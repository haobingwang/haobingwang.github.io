Windows 下使用 rsync

cwRsync 是 rsync for Windows，下载地址

下载解压放到 C 盘根目录 C:\cwRsync

然后在 PATH 中添加该路径，这样 cmd 中就可以直接运行 rsync ssh ssh-keygen 等命令了。

双击目录内的 C:\cwRsync\cwrsync.cmd ，会在当前目录生成目录： C:\cwRsync\home/USERNAME/.ssh

运行 ssh-keygen 命令就会在目录 C:\cwRsync\home\USERNAME.ssh 生成密钥，但是需要注意的是，这个密钥的权限有问题，在 ssh 使用时会提示：

这就需要安装 cygwin ，然后运行

这样，密钥就可以使用了。我们就可以不需要输入密码使用 rsync ssh 等命令了。

那么问题来了？

既然都安装了 cygwin，那干嘛不使用 cygwin 内安装的 rsync ssh ssh-keygen 呢？

对的，只需要将 C:\cygwin64\bin 添加到 PATH 中就可以了。

需要注意的是 cygwin 和 cwRsync 的密钥都是保存在各自的目录下面的即

C:\cygwin64\home\admin.ssh
C:\cwRsync\home\admin.ssh
