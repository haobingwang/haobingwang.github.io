cygwin

发表于 2016-09-08   |  

powershell ? 玩不来

win10 中有 ubuntu，win7 有 cygwin

安装 cygwin

选择从互联网安装, 在”User URL”处输入以下地址 http://mirrors.163.com/cygwin/

点击”Add”按钮,点击”下一步”进行安装

安装软件包

  • Net -> openssh
  • Net -> rsync

操作须知

在 cygwin 中：

  • C盘：/cygwin/c
  • D盘：/cygwin/d

修改 ssh 秘钥权限

CMD 中使用 cygwin 安装的包

将 C:\cygwin64\bin 添加到 PATH 中(最好加在windows前面，这样的话，有些相同的命令的话，是先执行cygwin的命令，而不是windows命令，比如find)

添加完毕，我们就可以在 cmd 中直接使用 C:\cygwin64\bin 内的 exe 了，比如 bash git rsync ssh ssh-keygen pwd 等等
