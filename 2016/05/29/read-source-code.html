阅读开源软件源代码的正确姿势

发表于 2016-05-29   |   分类于 learn   |  

以 react-router 项目为例

一、先Clone它

二、从Release页面找到它的早期版本

三、 从上面拿到它的提交号 6bf6d96，然后checkout到这次提交

或者在 sourcetree 中跳转

四、当前 release 为 v0.0,每个release 都是多个 commit 积累而成的。我们接着 checkout 到整个项目的第一个 commit 开始阅读代码

五、接着，再回过头去阅读
